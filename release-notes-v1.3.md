## DNS Hop v1.3

This release turns DNS Hop into a cleaner Windows and Linux desktop release while shipping the custom DNS workflow that has been in progress locally.

### Highlights
- Added a dedicated `Add Custom DNS` dialog for Classic DNS, DoH, and DoT endpoints
- Persisted custom DNS endpoints locally so they reload automatically on startup
- Fixed the Windows light-mode frame and corner presentation issues around the main window and overlays
- Added Linux AppImage packaging and CI smoke tests for Debian 12, Fedora 41, and Arch
- Refreshed the repo and website copy to present DNS Hop as a Windows and Linux desktop tool

### Assets
- `DNS-Hop-Setup-v1.3.exe`: Windows installer
- `DNS-Hop-Portable-v1.3.zip`: portable Windows build
- `DNS-Hop-AppImage-v1.3-x86_64.AppImage`: Linux AppImage
